B站视频实时在线人数排行榜

一个用于获取B站热门视频实时在线观看人数并生成排行榜的Python爬虫。

安装依赖

pip install requests beautifulsoup4

使用方法

直接运行脚本：
python video_ranking.py

程序会：
1. 获取B站热门视频列表
2. 获取每个视频的实时在线人数
3. 显示排序后的排行榜
4. 保存结果到 bilibili_online_ranking.json

输出示例：

=== B站视频实时在线人数排行榜 ===
排名  在线人数  UP主          标题
--------------------------------------------------
1.   12.3万+  UP主名称     视频标题
2.   1.2万+   UP主名称     视频标题
3.   1000+    UP主名称     视频标题
